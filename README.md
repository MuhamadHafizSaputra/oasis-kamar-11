## **Getting Started**
1. Clone repository
```
git clone https://github.com/MuhamadHafizSaputra/oasis-kamar-11.git
cd oasis-kamar-11
```
2. Install dependencies
```
cd sedulurkita

npm install
npm install @vis.gl/react-maplibre maplibre-gl
npm install axios
```
3. Jalankan project
```
npm run dev
```
4. Akses di browser<br>
Buka http://localhost:5173 untuk melihat aplikasi berjalan.
